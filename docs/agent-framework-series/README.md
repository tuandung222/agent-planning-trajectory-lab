# Agent Framework Series

Series này hướng dẫn sử dụng Microsoft Agent Framework theo lộ trình thực chiến, bám sát project hiện tại.

## Mục lục

1. [01 - Mental Model](./01-mental-model.md)
2. [02 - Clients and Providers](./02-clients-and-providers.md)
3. [03 - Tools and Safety](./03-tools-and-safety.md)
4. [04 - Workflows and Orchestration](./04-workflows-and-orchestration.md)
5. [05 - Observability and Trajectory](./05-observability-and-trajectory.md)
6. [06 - Notebook and Experimentation](./06-notebook-and-experimentation.md)
7. [07 - Production Checklist](./07-production-checklist.md)

## Cách học nhanh

- Bắt đầu với `01` và `02` để hiểu khung tư duy.
- Làm `03` và `04` để tự xây flow chạy được.
- Dùng `05` để thu trace data cho training.
- Dùng `06` nếu team DS/Researcher thích notebook.
- Chốt với `07` trước khi đưa vào production.
